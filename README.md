# PureMusic

Pure Music 

最新版已经基于跨平台重新开发，播放引擎也已经全量替换，暂未开源。  
目前支持Android、Mac，Windows、Linux 后续继续支持其他平台  
支持阿里、百度、onedrive、google drive及各种媒体服务源支持。
